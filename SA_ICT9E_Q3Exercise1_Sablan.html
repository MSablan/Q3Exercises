My First Bootstrap Page

HTML

HyperText Markup Languages

HTML is the common language when making websites. HTML will also have almost everything you will need when creating a website. All you need to do is put the exact code and you're good to go!

CSS

Cascading Style Sheets

CSS usually accompanies HTML. It is used to format the website. CSS also offers people the choice to have different kinds of styles for their webpage.

Bootstrap

Bootstrap Version 4

Bootstrap is basically another kind of code that accompanies both HTML and CSS. It is what makes websites responsive for smaller devices, like phones and tablets. Not only does it work on phones and tablets, it works on almost every device (Laptops, computers, TVs)
